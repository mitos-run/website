## Why mitos

Agent harnesses need fast, isolated environments where agents can read and write
files, install packages, and run untrusted code. Existing options force a trade:
speed without ownership, isolation without forking, or durability as someone
else's proprietary cloud.

<CardGrid>
  <Card title="N-way live fork" icon="random">
    Copy-on-write fork of a running microVM into parallel children. Fan out an
    agent into many attempts where the marginal cost of each is its dirtied

  </Card>
  <Card title="Milliseconds, not seconds" icon="rocket">

## Honest by construction

Every performance and security claim on this site traces to a reproducible
artifact in the project repository. If a number is not reproducible from
`bench/`, it does not appear here. Security findings block features.
